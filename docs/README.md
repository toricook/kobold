# Documentation Directory

This directory contains documentation for the Kobold project, published via GitHub Pages.

## Publishing to GitHub Pages

To enable GitHub Pages:
1. Go to your repository Settings
2. Navigate to Pages (under Code and automation)
3. Set Source to "Deploy from a branch"
4. Select your main branch and `/docs` folder
5. Click Save

Your documentation will be published at: `https://<username>.github.io/<repository>/`

## Structure

- `index.md` - Main documentation page
- `.nojekyll` - Prevents Jekyll processing (remove if you want to use Jekyll themes)

You can add more markdown files and organize them as needed.
