# Development Tools

Kobold provides several tools to streamline game development workflows.

## [Sprite Sheet Editor](sprite-sheet-editor.md)

Visual tool for creating sprite sheet configuration files.

**Features:**
- Load sprite sheet images
- Define grid parameters (tile size, spacing, margin)
- Visual grid overlay
- Interactive sprite selection
- Export JSON configurations
- Camera controls (pan, zoom)

**Use for:**
- Creating sprite sheet configs for animations
- Defining named regions
- Testing sprite layouts

**Quick Start:**
```bash
cd Kobold.SpriteSheetEditor
dotnet run
```

## [Experiments Sandbox](experiments.md)

Interactive testing environment for Kobold features.

**Features:**
- Rapid.NET UI for parameter tweaking
- Test procedural generation algorithms
- Visualize results in real-time
- No recompilation needed for parameter changes

**Use for:**
- Testing cellular automata configurations
- Experimenting with new features
- Prototyping game mechanics

**Quick Start:**
```bash
cd Kobold.Experiments
dotnet run
```
